Notes

RESTful route: A route that provides mapping between HTTP verbs (such as GET, POST, PUT, DELETE, and PATCH) to database actions (CRUD or INSERT, SELECT, UPDATE, DELETE).

Method (HTTP VERB) URL @app.route('') Route Handler Function Comments
GET '/' def index(): Show all users
POST '/friends' def create(): Create a new user from a form
GET '/friends/<id>/edit' def edit(id): Show the edit page with a single user
POST '/friends/<id> def update(id): Update a user
POST '/friends/<id>/delete def destroy(id): Delete a user
