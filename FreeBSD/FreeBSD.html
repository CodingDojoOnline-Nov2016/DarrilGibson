FreeBSD /ppc Project

Introduction

The Free BSD/ppc project pages contain information about the FreeBSD port to the PowerPC® architecture. As with the port itself, these pages are still a work in progress.

Status

The Free BSD/ppc port is still a Tier 2 platform. That means it is not being fully supported by our security officer, release engineers and toolchain maintainers.
